## About

This project was developed as part of the Web Design course at the University of Latvia. 
The website presents a personal overview of the University of Latvia's e-study environment and compares it with the E-klase platform previously used during school years. The project focuses on creating a clear, modern and user-friendly interface while applying web design and user experience (UX) principles.

## Design Concept

The website was designed with simplicity, readability and usability as the main priorities.
The visual style is based on blue, white and black colors, inspired by the University of Latvia's e-study environment. This creates a familiar and consistent appearance while maintaining a clean and modern interface.
The content is organized with images placed alongside descriptive text to improve readability and make navigation intuitive.

---

## User Experience

Special attention was given to:

- Easy navigation between sections
- Clear content hierarchy
- Responsive layout
- Readable typography
- Theme switching (Light / Dark mode)

---

## Technical Challenges

The biggest challenge during development was creating a responsive layout that works correctly across different screen sizes.
To solve this, the project uses:
- CSS Flexbox
- Media Queries
- Adaptive spacing and sizing
These techniques helped maintain usability on both desktop and mobile devices.
